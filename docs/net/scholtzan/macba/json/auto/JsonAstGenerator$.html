net.scholtzan.macba.json.auto

JsonAstGenerator

Related Doc: package auto

object JsonAstGenerator

Contains macros for automatically determining the structure of a passed object and it's corresponding JSON AST structure.

The JSON AST is based on json4s's JSON AST to provide compatibility with other libraries.
